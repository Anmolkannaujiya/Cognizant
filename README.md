# 📖 Modules

## 🎨 Design Patterns & Principles

### ✅ Exercise 1 — Singleton Pattern

**Project:** Logger Management System

**Description**

* Demonstrates the Singleton Design Pattern.
* Ensures only one Logger instance exists throughout the application.
* Used by multiple services such as User, Payment, and Order.

**Concepts Covered**

* Singleton Pattern
* Private Constructor
* Static Instance
* Global Access Point
* Object Reusability

---

### ✅ Exercise 2 — Factory Method Pattern

**Project:** Notification System

**Description**

* Demonstrates the Factory Method Design Pattern.
* Creates Email, SMS, and Push notifications through a Factory instead of directly instantiating objects.

**Concepts Covered**

* Factory Method Pattern
* Interfaces
* Polymorphism
* Loose Coupling
* Encapsulation of Object Creation
